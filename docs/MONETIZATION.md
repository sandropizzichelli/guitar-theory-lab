# Monetization

## Principles

- Keep core exploration usable in the Free tier.
- Put persistence, export, advanced comparison and personal libraries behind Pro.
- Avoid blocking essential learning workflows too early.
- Keep teacher and institution features separate from normal Pro use.

## Free

- Public tools
- Basic visualization
- Main calculations
- Base presets

## Pro

- Saved items
- Personal libraries
- PDF/image export
- Advanced presets
- Multiple comparison history
- Future advanced tools

## Teacher

- Pro features
- Teaching-material export
- Future class/student workflows

## Institution

- Conservatory, school and university licensing
- Centralized account management in a later phase
